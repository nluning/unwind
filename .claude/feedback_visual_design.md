name: Visual design preferences

Noor's brain wants "things to really stand for what they are" — blue should be a real, pretty blue. Aesthetics and visual clarity (neatness) are core values. The current theme colors were updated to be more saturated and vivid, but the overall UI still feels "too clinical."

**Why:** Noor is not the "less stimuli is better" type of neurodivergent user. They want visual personality and warmth, not sterile minimalism. That said, some themes (like Calm) should still cater to users who do prefer less stimulation.

**How to apply:** When building new UI (onboarding, etc.), lean toward design with warmth — consider rounded shapes, visual variety, subtle color use beyond just the primary accent. Revisit the clinical feel during the mobile styling pass (Stage 3 Step 15). Don't default to "grey box with one accent color" as the layout pattern.
